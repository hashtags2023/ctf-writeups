## Summary
Introductory room covering the fundamentals of computer networking — how devices communicate, IP vs MAC addresses, and basic network concepts like ping/ICMP.

## Key Concepts Learned
- What a network is and how the internet works
- IP Addresses (logical, changeable) vs MAC Addresses (hardware, spoofable)
- Identifying devices on a network
- Using ping and ICMP to check connectivity

## Notes
- MAC addresses are visible on shared networks like hotel Wi-Fi
- Modern OSes randomize MACs per network for privacy
- MAC spoofing can be used to bypass filtering or impersonate devices
